# New Employee Security Onboarding Checklist

**Get new staff set up securely from day one**

---

## üéØ Purpose

This checklist ensures every new employee understands security basics and has secure access to systems before they start work.

**Why it matters:**
- New employees are security weak points if not trained
- Setting up security from day one is easier than fixing later
- Compliance requirement (GDPR, Cyber Essentials)
- Protects the business and the employee

**Time needed:** 1-2 hours total, spread over first week

## üìã Day One Checklist

### ‚úÖ Welcome and Setup (First Hour)

**Account access:**
- [ ] Help log into email first time
- [ ] Set up MFA (Authenticator app)
- [ ] Verify email sending/receiving works
- [ ] Show how to access other systems
- [ ] Demonstrate password manager use

---

### ‚úÖ Security Briefing (30-45 minutes)

**Schedule dedicated time for this - don't rush it**

**Part 1: Why Security Matters (5 minutes)**
- We handle customer personal data (GDPR responsibility)
- Cyber attacks target small businesses
- Everyone plays a role in security
- It's easier than it sounds

**Part 2: Key Security Rules (15 minutes)**

**Passwords:**
- [ ] Explained: Use password manager for everything
- [ ] Demonstrated: How to use password manager
- [ ] Practiced: Generate a secure password together
- [ ] Set up: Install password manager on their devices
- [ ] Emphasized: Never share passwords, even with colleagues

**Multi-Factor Authentication (MFA):**
- [ ] Explained: What MFA is and why we use it
- [ ] Demonstrated: How Authenticator app works
- [ ] Practiced: Set up MFA on their email
- [ ] Reminded: Keep phone secure (it's now your "key")

**Device Security:**
- [ ] Keep devices locked when not in use
- [ ] Don't leave phone in unlocked van
- [ ] Don't share device PIN with family/friends
- [ ] Report lost/stolen devices IMMEDIATELY

**Email and Phishing:**
- [ ] How to recognize phishing emails
- [ ] Never click suspicious links
- [ ] Never download unexpected attachments
- [ ] If in doubt, ask before clicking

**Part 3: Customer Data Handling (10 minutes)**

**What is customer data:**
- Names, addresses, phone numbers
- Photos of their homes
- Payment information
- Gas certificates

**How to handle it:**
- [ ] Only use it for the job
- [ ] Don't take customer details home
- [ ] Don't discuss customers where others can overhear
- [ ] Don't screenshot and text customer details
- [ ] Use Tradify for job info, not personal notes
- [ ] Upload photos to Tradify, delete from phone

**Part 4: What to Do If Something Goes Wrong (5 minutes)**

**Report immediately if:**
- You lose your phone or laptop
- You click a suspicious link
- You think your account is compromised
- You see something unusual on systems
- You accidentally send data to wrong person

**Who to contact:**
- [ ] Provided: Emergency contact card
- [ ] Saved: Numbers in their phone
- [ ] Emphasized: Call anytime, day or night
- [ ] Reassured: No punishment for honest mistakes if reported

---

## üìã Offboarding Checklist

**When employee leaves, complete within 24 hours:**

### ‚úÖ Access Revocation
- [ ] Email account disabled
- [ ] Password manager access removed
- [ ] Tradify / job management access removed
- [ ] Xero access removed (if applicable)
- [ ] Any other system access removed
- [ ] Company Google/Apple account removed from device
- [ ] Office keys / access cards returned

- [ ] Company phone returned (if provided)
- [ ] Laptop returned (if provided)
- [ ] Any other company property returned
- [ ] Factory reset performed on returned devices
- [ ] Find My Device disabled

### ‚úÖ Data Protection
- [ ] Personal data removed from any personal devices (if BYOD)
- [ ] Work apps uninstalled from personal devices
- [ ] Confirmed no company data kept on personal systems
- [ ] Customer data access logged and reviewed

### ‚úÖ Shared Credentials
- [ ] Change any shared passwords they had access to
- [ ] Update WiFi password if they knew it
- [ ] Update any shared accounts

### ‚úÖ Documentation
- [ ] Offboarding checklist completed

- [ ] Confirmation employee understands ongoing confidentiality obligations

## ‚ùì Common Questions

**"What if new employee resistant to password manager?"**
- Show how it makes life easier (one master password instead of dozens)
- Demonstrate on your own account
- Help them set it up properly
- Give it 2 weeks before judging
- Make it non-negotiable for company systems

**"What if they forget their master password?"**
- Password managers have recovery options
- Document recovery process during setup
- Keep emergency contact updated

**"Do we really need MFA for everyone?"**
- Yes, especially on email and financial systems
- It's the single most effective security control
- Takes 30 seconds to set up, saves hours after a breach

**"What about contractors or temps?"**
- Same security requirements
- May provide own devices (must meet security standards)
- Remove access immediately when contract ends
